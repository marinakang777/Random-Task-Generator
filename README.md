## Описание

Приложение "Random Task Generator" разработано с использованием Python и библиотеки Tkinter для создания графического пользовательского интерфейса (GUI). Оно позволяет пользователю:

*   Генерировать случайные задачи из предопределенного списка.
*   Просматривать историю сгенерированных задач.
*   Фильтровать задачи по их типу (например, "Учёба", "Спорт", "Работа", "Личное").
*   Добавлять собственные задачи с указанием категории.
*   Сохранять весь список задач и историю их генерации в файл `history.json` для последующей загрузки.

Приложение использует:
*   `tkinter`: Для создания GUI.
*   `random`: Для выбора случайных задач.
*   `json`: Для сохранения и загрузки данных.
*   `os`, `datetime`: Для работы с файловой системой и датой/временем.

## Использование

1.  **Генерация задачи:** Нажмите кнопку "Сгенерировать задачу". Случайная задача будет отображена в центральной части окна.
2.  **Фильтрация:** В выпадающем списке "Фильтр по типу" выберите нужную категорию (или "Все типы"), чтобы генерировать задачи только из выбранной категории. При выборе типа задача будет автоматически перегенерирована.
3.  **Добавление задачи:** Нажмите кнопку "Добавить задачу". Откроется новое окно, где вы можете ввести описание новой задачи и ее тип. Нажмите "Добавить", чтобы сохранить задачу.
4.  **История:** Все сгенерированные задачи с временной меткой и типом отображаются внизу в виде таблицы.
5.  **Сохранение:** История и список задач автоматически сохраняются в файл `history.json` при закрытии приложения. При следующем запуске данные будут загружены.

## Примеры использования (Тесты)

*   **Базовая генерация:**
    *   Запустите приложение.
    *   Нажмите "Сгенерировать задачу". Убедитесь, что отображается текст задачи.
    *   Проверьте, что запись появилась в таблице истории.
*   **Фильтрация:**
    *   Выберите тип "Спорт" в фильтре.
    *   Нажмите "Сгенерировать задачу". Убедитесь, что задача относится к спорту.
    *   Выберите "Все типы" и сгенерируйте задачу. Убедитесь, что снова munculлась задача другого типа.
*   **Добавление задачи:**
    *   Нажмите "Добавить задачу".
    *   Введите "Помыть посуду" как описание и "Быт" как тип.
    *   Нажмите "Добавить".
    *   Выберите "Все типы" в фильтре.
    *   Нажмите "Сгенерировать задачу". Убедитесь, что новая задача ("Помыть посуду") может быть сгенерирована.
    *   Закройте приложение и запустите снова. Проверьте, что "Помыть посуду" все еще есть в общем списке задач.
*   **Пустой вот:**
    *   Нажмите "Добавить задачу".
    *   Попробуйте оставить поле "Описание задачи" пустым и нажать "Добавить".
    *   Убедитесь, что появляется предупреждение и задача не добавляется.
